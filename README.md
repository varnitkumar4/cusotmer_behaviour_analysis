# Customer Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior to uncover patterns, trends, and business insights using Python, SQL, and Power BI.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL
- Power BI

## Project Workflow
1. Data Cleaning & Preprocessing (Python)
2. Exploratory Data Analysis (EDA)
3. SQL Queries for Business Insights
4. Dashboard Creation using Power BI

## Key Insights
- Identified top-performing product categories
- Analyzed customer spending behavior
- Observed purchasing trends
- Found opportunities for customer retention
